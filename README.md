# corporatools
Various stand-alone scripts for parsing different corpora

## Arramooz
[Arramooz](https://github.com/linuxscout/arramooz) is an Arabic dictionary. 

### noun_to_csv
This script builds a csv of nouns, and pulls information from the wordfreq csv and the sqlite noun database from Arramooz.


## For CELEX
[CELEX](http://celex.mpi.nl/) is a database containing morphological information 

### DISC_to_IPA
This script takes DISC transcription from CELEX and breaks it down by syllable and converts it to IPA
